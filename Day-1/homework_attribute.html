Homework: Explore HTML Attributes

Your task is to create a webpage with the following elements and explore how attributes modify their behavior:

  1. Create an <img> tag and use the src attribute to point to an image.
  2. Use the alt attribute for your image to add alternative text.
  3. Create a link with the <a> tag and use the href attribute to link to an external website.
  4. Use the title attribute on your link to show extra information when hovering over it.
  5. Add a <button> tag with a type attribute set to "submit".
  6. Create a text input using the <input> tag with the placeholder attribute.

Submit your assignment by creating a new file called attributes_homework.html and implementing the above points!
